# Smart University Parking System 🚗

An intelligent solution designed to streamline parking management within university campuses. This system optimizes space utilization and improves the experience for students and faculty.

## ✨ Key Features
* **QR-Based Entry:** Secure and fast check-in/out using QR codes.
* **Real-time Monitoring:** Live tracking of available parking slots.
* **Priority System:** Dedicated reservation logic for faculty and staff.
* **User Dashboard:** Simple interface for students to view availability.

## 🛠️ Tech Stack
* **Frontend:** Flutter (Mobile App)
* **Backend:** Firebase / Node.js (Real-time Database)
* **Tools:** Figma for UI/UX design.

## 🛡️ Security Features
* Encrypted user data and booking records.
* Secure authentication system.
